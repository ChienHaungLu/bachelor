# Introduction

## Algorithm
演算法主要為大學部必修課程，存放過往的撰寫作業或專案等

## C_language_Homework
主要為大學部必修課之C語言課程功課存放區

## ComputerGraph
主要為大學部選修課程，存放為電腦圖相關的專案製作，過程中有使用到OpenGL的API

## Database
主要為大學部必修課程，資料庫系統為專案存放區，有與校外公司合作取得商品或合作資料等，因保密協議不提供商品客戶資料

## Digital-System
主要存放大學部之數位系統課程，本課程作業是透過**VHDL**硬體描述語言進行撰寫，並在QuartusII之EDA設計軟體上進行測試，確保功能的正確性

## Senior Project
此為大學部畢業專題檔案存放區，主要的深度學習框架為Pytorch，資料集來源皆和國外作者簽署同意書取得授權!
