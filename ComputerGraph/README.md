## HW4
主題: free fall  
說明: 非剛性(non-rigid)六面體體，從靜止狀態自空中自由落下(S=0.5*a*t^2,請自行選擇適當a值)，觸及地面後產生垂直方向產生擠壓->回復形變 與 水平方向產生剪力應變(Shear，詳見課程PPT) 至少三次且幅度漸次縮小。另建置含兩個選項的彈跳式選單(pop menu)，一者為重新執行；另者為結束程式。
